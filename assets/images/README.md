# Photos

Put travel photos in this folder.

Recommended structure:

```text
assets/images/
├─ uzbekistan/
│  └─ samarkand-blue-morning.jpg
├─ hungary/
│  └─ budapest-river-walk.jpg
└─ japan/
   └─ kyoto-autumn-lane.jpg
```

Use photos in posts like this:

```markdown
![撒馬爾罕藍色清晨](/assets/images/uzbekistan/samarkand-blue-morning.jpg)
```

For faster loading, resize photos to around 1600px to 2000px wide before uploading.
